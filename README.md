# calculador precios

Price calculator built with JavaScript and a simple web interface.

## Tech Stack

HTML5, CSS3, JavaScript and Bootstrap

## Getting Started

```bash
git clone https://github.com/MagdaIG/calculador-precios.git
cd calculador-precios
Open index.html in your browser
```
